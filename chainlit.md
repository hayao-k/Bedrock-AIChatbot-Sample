# AI Chatbot powered by Amazon Bedrock 🚀🤖

## About
Chainlit + LangChain + Bedrock なサンプルアプリケーションです。  
Settings panel からモデルやパラメーターを設定できます。

### 選択可能なモデル

### パラメーター
**Temperature:** デフォルト: 0.3  

**Max Token Size:** デフォルト: 1024
